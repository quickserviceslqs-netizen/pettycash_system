Important Notes
  • Usernames: Auto-generated from first and last name (e.g., JohnDoe, JaneSmith)
  • Passwords: Users set their own password via invitation link
  • Duplicate Check: System prevents duplicate emails
  • Organizations: Company, Department, Branch must already exist in system
  • Apps: Use app names: treasury, workflow, reports, settings
  • Roles: Must be one of: REQUESTER, APPROVER, FINANCE, ADMIN
CSV Template Format
email first_name last_name role company_name department_name branch_name assigned_apps
john.doe@example.com John Doe REQUESTER Quick Services LQS Finance Head Office treasury,workflow
jane.smith@example.com Jane Smith APPROVER Quick Services LQS Finance Head Office treasury,workflow,reports

Result: Username will be JohnDoe and JaneSmith. If duplicates exist, system adds number suffix (e.g., JohnDoe1, JohnDoe2).
