Important Notes
  • Usernames: Auto-generated as FirstInitial.LastName (e.g., A.Cheloti for Amos Cheloti)
  • Passwords: Users set their own password via invitation link
  • Duplicate Check: System prevents duplicate emails
  • Multi-Company: Template includes ALL companies, branches, departments, regions
  • Exact Names Required: Company, Department, Branch must EXACTLY match (copy from template list)
  • Region Filtering: Use region_name to filter branches by region
  • Apps: Use app names: treasury, workflow, reports, settings
  • Roles: Must be one of: REQUESTER, APPROVER, FINANCE, ADMIN
CSV Template Format
email first_name last_name role company_name region_name branch_name department_name assigned_apps
amos.cheloti@example.com Amos Cheloti REQUESTER Quick Services LQS East Africa Nairobi Branch Finance treasury,workflow
jane.doe@example.com Jane Doe APPROVER Quick Services LQS East Africa Mombasa Branch Operations treasury,workflow,reports

Result: Username will be A.Cheloti and J.Doe. If duplicates exist, system adds number suffix (e.g., A.Cheloti1, A.Cheloti2).
Note: Region helps filter branches when multiple branches have same name.
